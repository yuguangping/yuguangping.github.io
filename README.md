# Yuguangping Personal Homepage

This repository contains the source for `https://yuguangping.github.io`.

It is built with Jekyll and based on [RayeRen/acad-homepage.github.io](https://github.com/RayeRen/acad-homepage.github.io).

## Main Files

- `_config.yml`: site metadata and author profile.
- `_pages/about.md`: homepage content.
- `_data/navigation.yml`: top navigation.
- `images/`: avatar, favicon, and project images.
- `CUSTOMIZE.md`: practical checklist for editing and publishing.

## Local Preview

Install Ruby and Bundler first, then run:

```powershell
bundle install
bundle exec jekyll serve
```

Open `http://127.0.0.1:4000`.
